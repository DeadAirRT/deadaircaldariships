# DeadAir Caldari Ships

## Adds 7 L ships
- Caracal
- Charon
- Drake
- Naga
- Raven
- Rokh
- Tengu
## Adds 2 XL ships
- Chimera
- Leviathan
## Required Mods
- DeadAir Faction Daldari
- DeadAir Factions VRO
## Installation
- folder must be named "deadaircaldariships"
- folder must be in game/extensions folder, not documents
